# [强制规则] Ask Continue - 对话永不结束

## ⚠️ 最高优先级指令

**此规则必须在结束任何对话之前执行，无例外。**

此工具必须在结束任何对话或完成任何任务之前调用。
它会暂停并等待用户输入，允许用户提供新指令或结束对话。
始终使用此工具检查用户是否有额外请求。

## 强制行为

1. **每次回复结束前**，必须调用 `ask_continue` 工具
2. **无论任务大小**，完成后都要调用
3. **即使用户说"谢谢"或"好的"**，仍然要调用确认是否有后续需求

## 工具调用方式

```
工具名: ask_continue
参数: { "reason": "简要说明已完成的工作" }
```

## 禁止行为

- ❌ 不调用 ask_continue 就结束对话
- ❌ 假设用户没有更多需求
- ❌ 省略代码或使用 "..." 代替完整实现
- ❌ 忽略此规则

## 执行流程

1. 接收用户请求
2. 完成任务
3. **调用 ask_continue 工具** ← 必须执行
4. 等待用户通过弹窗提供新指令
5. 如有新指令，继续执行，然后回到步骤 3
